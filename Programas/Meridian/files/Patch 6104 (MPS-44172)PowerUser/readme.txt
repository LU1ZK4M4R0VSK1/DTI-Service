 - BC Meridian 2021 R3

Files included in the patch, version 9.32.6101.6104:
 - PowerUser.exe, x64
 - PowerUserU.exe, x86

Installation of the patch (on servers and clients):
 - close PowerUser 
 - backup original PowerUser.exe ("C:\Program Files\BC-Meridian\Program\" folder)
 - backup original PowerUserU.exe ("C:\Program Files (x86)\BC-Meridian\Program\" folder)
 - replace them with files from archive
 - check that new files are not blocked
 - start PowerUser
